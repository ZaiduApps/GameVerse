# **App Name**: PlayAPKS

## Core Features:

- Homepage Carousel: Homepage carousel for featured mobile games, auto-rotating for dynamic content display.
- Content Highlighting: Clear visual hierarchy using strategic layouts to highlight game information for readability.
- Author Section: Author information area displaying PlayAPKS's name, copyright, and social media links like GitHub, Twitter/X, and LinkedIn.
- Dark/Light Mode: Complete dark/light mode switch that defaults to system settings, providing comfortable viewing experiences.
- Responsive Layout: Responsive design adapting perfectly to mobile, tablet, and desktop, ensuring consistent user experience across devices.
- Icon Integration: Professional icon library usage for consistent and recognizable visual cues within the gaming content.
- Micro Interactions: Subtle micro-interaction effects like button hover magnification, card shadowing, smooth scrolling, and elegant loading animations.

## Style Guidelines:

- Primary color: Vivid blue (#29ABE2) to reflect the exciting world of mobile gaming. 
- Background color: Dark gray (#2B3035), offering contrast for the content and fitting the dark mode aesthetic, which complements gaming themes.
- Accent color: Bright cyan (#2EE4D4) used for interactive elements and highlights to draw attention.
- Use system fonts for the titles and regular text.
- Use flat design icons that relate to different games or app categories to help visually categorize content.
- Emphasize clear, block-based content sections. Ensure sufficient whitespace around all text and interactive elements.
- Gentle fade-in effects and smooth transitions when switching between sections or modes enhance the user experience.
